# Самостійна робота 

## Написати чат для локальної мережі з використанням сокетів.

- Обовязково має бути сервер через який будуть проходити всі повідоилення.
- Сервер має однозначно ідентифікувати клієнта (IP ...)
- Передбатити можливість відправки повідомлень індивідуально так і для всіх
- Передбачити можливість створення індивідуального каналалу в обхід сервера. (повідомлення відправляються напряму)
- Реалізація може бути як (Console,WPF)
